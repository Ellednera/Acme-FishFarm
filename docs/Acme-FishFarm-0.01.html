NAME

Acme::FishFarm - A fish farm that has automated systems

VERSION

Version 0.01

SYNOPSIS

    use Acme::FishFarm;

EXPORT

None by default

CREATION RELATED METHODS

new ()

Creates a new fish farm.

install ( systems => [...] )

Installs the specified systems to your fish farm. This is done by setting the systems key.

The values for systems include:

  • water_level

  • feeder

  • water_filter

  • water_monitor

    #=item * oxygen_level

To install all the systems in one go, use the install_all_systems method instead.

install_all_systems ()

Installs all the systems available in the fish farm. See install method for the available systems

WATER LEVEL MAINTAINER SUBROUTINES/METHODS

check_water_level ()

Returns the current water level in your pond, tank, aquarium or whatever.

AUTOMATED FEEDER SUBROUTINES/METHODS

check_feeder () ??? maybe for refilling :)

Returns the amount (volume) of fish food in the feeder.

This is only useful if you want to see how much fish food is still there. This is not an important feature of this module

WATER FILTRATION SUBROUTINES/METHODS

check_water_filter () ??? maybe just to inspect for fun :)

Returns the amount of waste in the filtering cylinder.

WATER CONDITION MONITORING SUBROUTINES/METHODS

check_water_monitor ()

Returns a hash of the current condition of the water. The hash will include the following keys:

1. pH

The values are in the range of 1-14

2. temperature

The ranges of values are between 0 and 50 degree celciuls

3. DO

The range of values are between 0 and 10 mg/L

4. turbidity

The range of values are between 0 and 300 ntu.

OXYGEN LEVEL MAINTAINER SUBROUTINES/METHODS

check_oxygen_level ()

This is the same as checking the DO content through the check_water_filter method.

Either this part of the DO part might be removed
