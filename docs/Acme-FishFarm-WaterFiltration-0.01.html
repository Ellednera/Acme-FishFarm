NAME

Acme::FishFarm::WaterFiltration - Water Filter for Acme::FishFarm

VERSION

Version 0.01

SYNOPSIS

    use Acme::FishFarm::WaterFiltration;

    my $foo = Acme::FishFarm::WaterFiltration->new();
    ...

EXPORT

None

DESCRIPTION

This module assumes a cool water filter with a filtering cylinder constantly filtering water in the tank. It has inlet, outlet and a drainage valves. The drainage valve is only opened when the cleaners are switched on automatically to remove waste from the cylinder. To be honest, those cleaners look more like spatulas to me :)

This feature is based on the water filter found here

CREATION SUBROUTINES/METHODS

install ( $waste_threshold )

Installs a water filtration system.

waste_threshold is optional and the default value is 75.

The $waste_threshold is the maximum limit of waste in the cylinder. When this count is hit, it will turn on the cleaners / spatulas or whatever it's called :).

We'll just assume the range of 0 to 100 for this waste counter.

WASTE LEVEL DETECTING SUBROUTINES/METHODS

read_waste_counter ()

Gets the current waste count.

get_waste_counter_threshold ()

Gets the waste count threshold.

set_waste_counter_threshold ()

Sets the waste count threshold.

CLEANING RELATED SUBROUTINES/METHODS

turn_on_spatulas ()

Turns on the cleaning mechanism ie spins the spatulas :)

The spatulas will scrub the waste off the filter lining. This means the waste count will drop.

The spatulas should be turned on until the waste count is 0 before turning off. This should be implemented in the main script.

turn_off_spatulas ()

Turns off the spatulas and watch the waster counter rise again. :)
